# Follow Button

The Follow Button component allows users to follow, unfollow, mute, or block another Ethereum address or ENS name.

## Prerequisites

Before using the Follow Button component, ensure you have setup the [Transaction Modal](./transaction-modal.mdx) component.

## Add to your project

```tsx copy
import { FollowButton } from '@encrypteddegen/identity-kit'

export default function Home() {
  return <FollowButton lookupAddress="0x1234...abcd" connectedAddress="0xabcd...1234" />
}
```

## Parameters

| Parameter             | Description                                                     | Required | Default Value |
| --------------------- | --------------------------------------------------------------- | -------- | ------------- |
| `lookupAddress`       | The Ethereum address or ENS name to follow or unfollow.         | Yes      | -             |
| `connectedAddress`    | The Ethereum address of the currently connected user.           | No       | -             |
| `disabled`            | Whether the button is disabled.                                 | No       | false         |
| `onDisconnectedClick` | Function to call when the button is clicked while disconnected. | No       | -             |
| `className`           | Additional CSS class names to apply to the button.              | No       | -             |
| `customLoader`        | Custom loader component to display while loading.               | No       | -             |
| `props`               | Additional props for the `<button>` element.                    | No       | -             |
