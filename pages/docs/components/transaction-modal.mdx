# Transaction Modal

The Transaction Modal component allows users to initiate and manage on-chain transactions, providing a user-friendly interface for interacting with blockchain operations. Transactions are stored in local storage until they are completed.

## Add to your project

<Steps>
  Before using the Transaction Modal component, ensure you have setup the [Wagmi](https://wagmi.sh/) library, Viem, [Tanstack Query](https://tanstack.com/query) client, and have wrapped your application with the TransactionProvider
  
  ### Install Wagmi and Viem
  
  ```sh npm2yarn copy
  npm install wagmi viem@2.x @tanstack/react-query
  ```
  
  ### Setup Wagmi Config
  
  EFP currently supports
  
  ```tsx copy
  import { http, createConfig } from 'wagmi'
  import { chains as EIK_SUPPORTED_CHAINS, transports as EIK_SUPPORTED_TRANSPORTS } from 'ethereum-identity-kit'
  
  export const config = createConfig({
    chains: [
      ...EIK_SUPPORTED_CHAINS,
      /* add more chains here */
    ],
    transports: [
      ...EIK_SUPPORTED_TRANSPORTS,
      /* add more transports here */
    ],
  })
  ```
  
  ### Setup Providers
  
  Wrap your application with the WagmiProvider and TransactionProvider
  
  ```tsx copy
  /* your imports */
  import { WagmiProvider } from 'wagmi'
  import { config } from './wagmi-config'
  import { TransactionProvider } from 'ethereum-identity-kit'
  import { QueryClient, QueryClientProvider } from '@tanstack/react-query'
  
  const queryClient = new QueryClient()
  
  export default function App() {
    return (
      <WagmiProvider config={config}>
        <QueryClientProvider client={queryClient}>
          <TransactionProvider>{/* other providers & components */}</TransactionProvider>
        </QueryClientProvider>
      </WagmiProvider>
    )
  }
  ```
  
  ### Add Component

Modal can be added anywhere in your application, but it is recommended to add it to the root (or as close to the root as possible) of your application.

```tsx copy
import { TransactionModal } from '@encrypteddegen/identity-kit'

export default function Home() {
  return <TransactionModal className="custom-transaction-modal" />
}
```
